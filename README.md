# myt: mytail and exploration into writing my own tail

# Usage

```
myt -n <integer> -r -d <filename>
```

where:

  `-r` use the recursive algorithm otherwise iterative
  `-d` debug
  `<filename>` is a text file to tail otherwise stdin


# Example

```
bash$ myt -n 1 -r myt.c
}
bash$ 
```
